# TaskStudent App

Aplicación web para la gestión de tareas académicas, desarrollada de forma colaborativa utilizando Git y GitHub.

## Objetivo del proyecto

Desarrollar una aplicación sencilla que permita a los estudiantes organizar sus actividades escolares, aplicando control de versiones y trabajo en equipo mediante el uso de ramas.


## Funcionalidades principales
- ➕ Agregar nuevas tareas
- ✅ Marcar tareas como completadas
- 🗑 Eliminar tareas
- 📊 Visualizar lista de tareas pendientes
- 🔢 Contador dinámico de tareas

## Tecnologías utilizadas
- HTML (Frontend)
- CSS (Diseño)
- JavaScript (Backend simulado)
- Python (Pruebas)

## Control de versiones
Se utilizaron ramas:
- frontend = desarrollo de la interfaz
- backend = lógica de la aplicación
- diseño = estilos visuales
- tester = pruebas en Python
- documentacion = elaboración del README
